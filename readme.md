# crop6524

Crop images to 65:24 aspect ratio.


## install

```
$ cargo install --locked --git https://github.com/sakti/crop6524.git
```

## usage

```
$ crop6524 --help
Usage: crop6524 [OPTIONS] --target <TARGET>

Options:
  -t, --target <TARGET>
  -o, --output <OUTPUT>  [default: output.jpg]
  -h, --help             Print help
  -V, --version          Print version

```
